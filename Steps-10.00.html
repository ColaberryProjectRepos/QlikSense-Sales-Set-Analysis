Create the KPIS

After downloading the Vizlib extension you should see it on the Custom objects:

Final Visitation for this KPI should look like this:

Gross Profit Margin %

We will need to create 6 Layers :

Title:

1-Click on Custom objects and then drag and drop #1 Vizlib KPI to the main sheet

2- Click on Add Layer then click on Text to add the Title

3-Write Gross Profit Margin on the Text Box area   

4-Adjust the Title to be on the top left corner and chose the format you like(Font and Size, and color)

 

 

Icon:

1- Click on Add Layer then click on Icon to add the icon, select trending Icon

2-Adjust to Icon to be on the top left corner and chose the wright format 


Actual KPI

1- Click on Add Layer then click on Text

2-Adjust to Tex to be on the middle of the KPI and chose the wright format

3-On the text box write the formula:

=Sum({<Year = {$(=Max(Year))}>} Profit)/Sum({<Year = {$(=Max(Year))}>} Sales)

4- On the Format chose Number and 100.00% and on the Negative format -100.00%

 

5-On the color area select fx and type the formula:

=if ((Sum({<Year = {$(=Max(Year))}>} Profit)/Sum({<Year = {$(=Max(Year))}>} Sales)) >'10%', green(), red())

Target

For the target we need two text boxes one for title Target 10% and one for the calculation

First Title:

1- Click on Add Layer then click on Text

2-Adjust to Tex to be on the middle of the KPI under the actual KPI and chose the wright format

3-write Target 10% on the text box

Calculation

1- Click on Add Layer then click on Text

2-Adjust to Tex to be on the middle of the KPI under front-off Target 10 and chose the wright format

3-On the Text box write the formula:

=Sum({<Year = {$(=Max(Year))}>} Profit)/Sum({<Year = {$(=Max(Year))}>} Sales)/10)-1 /100 )* 100

 4- On the Format chose Number and 100.00% and on the Negative format -100.00%

5-On the Font Color select fx and type the formula:

=if (((Sum({<Year = {$(=Max(Year))}>} Profit)/Sum({<Year = {$(=Max(Year))}>} Sales)) -0.1) >'0', green(), red())

Trend:

1-      Click on Add Layer then click on Linechart


2-      Chose  MonthYear as a Dimension and sort by Dimension




3-      On the Measure type the formula:=Sum({<Year = {$(=Max(Year))}>} Profit)/Sum({<Year = {$(=Max(Year))}>} Sales)

4-      On the Line Style select Area


 

Note: make sure to name all your layers. 



Repeat the same steps for the other two KPIS

 

Customer Satisfaction %

Formula:

=(Count([Order ID])-Count(ReturnedRegion))/ Count([Order ID])

Note: You can use Variables for the formula if you like ex:

Vsatisfatory  =(Count([Order ID])-Count(ReturnedRegion))/ Count([Order ID])

To create your variables click on the bottom left where it says X and create you variables

Color formula for the main KPI:

=if ((Vsatisfatory) >'97%', green(), red())

Target is 97%

Formula is

=((Vsatisfatory/97)-1 /100 )*100

Color formula for the target :

=if ((Vsatisfatory) >'0', green(), red())

Trend formula :

=(Count({<Year = {$(=Max(Year))}>} [Order ID])-Count({<Year = {$(=Max(Year))}>} ReturnedRegion))/Count({<Year = {$(=Max(Year))}>} [Order ID])

 

Discounts %:


Formula:

=Sum({<Year={$(=Max(Year))}>}[Discounted Amount])/Sum({<Year={$(=Max(Year))}>}Sales)

Target is 8%

Formla:

=((Vdiscount/8)-1 /100 )*-100

Color for formula:

=if ((((Vdiscount/8)-1 /100 )*-100) >'0', green(), red())

Trend formula:

=Sum({<Year={$(=Max(Year))}>}[Discounted Amount])/Sum(Sales)
